This is the open-source toolchain Brain Brigade runs on. Every entry below is a self-contained Claude skill, plugin, or tool you can install, fork, or eval against your own workflow. No paywall. No invite. No middleman.

Tags map each skill to one of the Octo Agent arms — that's how we route them internally. You don't have to use the routing; the skills work standalone.

02 CLI / Developer Tools

Puppeteer

Headless Chromium with full Chrome DevTools Protocol access. The actual Loki substrate — exposes Console.messageAdded, Network.requestWillBeSent, and Runtime.evaluate() for instrumenting both dev-server logs and rendered DOM. Active, Google-backed.

loki osint CDP
npm install puppeteer
docs ↗

Carbonyl ⚠ archive

Chromium-in-terminal — technically impressive (custom Skia shim, 60 FPS Unicode-block rendering) but abandoned since Feb 2023. View-only; no CDP, no programmatic hooks. Interesting demo, not a Loki substrate. Use Puppeteer instead.
